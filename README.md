# Classical Cipher Toolkit

A Python project implementing classical cryptographic ciphers from scratch.

## Features

### Implemented
- Caesar Cipher
- Vigenère Cipher

### Planned
- Affine Cipher
- Playfair Cipher
- Hill Cipher
- Frequency Analysis
- Caesar Cipher Cracker
- Command Line Interface Improvements
- Unit Tests

## Project Structure

```
classical-cipher-toolkit/
│
├── main.py
├── ciphers/
│   ├── __init__.py
│   ├── caesar.py
│   └── vigenere.py
```

## Example

Encrypt using a Caesar cipher:

```text
Choose cipher: 1

Message: HELLO
Shift: 3
```
